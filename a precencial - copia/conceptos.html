definicion
Document.queryselectorAll() El método querySelectorAll() de un Element devuelve una NodeList estática (no viva) que representa una lista de elementos del documento que coinciden con el grupo de selectores indicados.
Document.getElementById() El método getElementById permite, como su nombre indica, seleccionar un elemento del documento por medio del valor del atributo id que se le haya asignado.
Documentqueryselector() querySelector() Devuelve el primer elemento del documento (utilizando un recorrido primero en profundidad pre ordenado de los nodos del documento) que coincida con el grupo especificado de selectores.
Document.getElementByClassName() getElementsByClassName() Retorna un objecto similar a un array de los elementos hijos que tengan todos los nombres de clase indicados.
this Se usa this cuando un atributo está oculto por una declaración de variable o parámetro. Por ejemplo: En el segundo constructor titulo y autor están ocultos por los parámetros con el mismo nombre.
evento Eventos: Son aquellas acciones mediante las cuales el objeto reconoce que se está interactuando con él
